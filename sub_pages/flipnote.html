Flipnote

Now we just have to put the show on for real.

More details

Alongside my studies, I have a passion for theater, and I was asked to develop software to plan a complete production. At the same time, the rise of no-code development was just beginning, and I wanted to give it a try, which is why I used Lovable to see what kind of results it would yield. In the end, the result turned out pretty well, and I continue to improve it constantly. Even though I didn’t write most of the code myself, I fully understand Node.js and have even modified parts of it myself.

This web app, with an interface inspired by After Effects, was originally designed solely for creating 2D and 3D animations, but later on, I wondered if it was possible to add more features, which led to a fireworks simulation mode, a DMX control mode with a real DMX communication system accessible via the browser, a 3D model manager, a drone control system, and a universal mode that combines all the others, with a physics-based or logic-based option similar to Cisco Packet Tracer.

This project is still under development but aims to be as accessible, versatile, and user-friendly as possible; it is also possible to add your own 3D models, spotlights, or fireworks. The show can then be exported as a PDF, JPG, or MPEG to keep a record of it. Additionally, if a user uses their own files, they will be saved with the show and can be shared directly.

This project is still under development but may be accessible at any time on the website or the github repository
