Humanoid

Robot

a friend of plastic and wires

More details

Ever since I was little, I've loved robot movies like Chappie and Real Steel. I find it fascinating to replicate human behavior with metal and lines of code.

In 2021, a friend and I built a prototype animatronic eye, designed by Will Cogley. That's when my desire to build things really took off, and I started learning about modeling and 3D printing.

In 2022, I searched for content on robotics and 3D printing, and I discovered INMOOV, a robot that can be built and printed in 3D. I started printing and assembling the parts so that it now looks like a humanoid.

After that, I started looking into programming, wondering how it was possible to code intelligence with code and whether it could be possible. In 2023, Meta released Ollama, a local AI system, which was exactly what I needed, so I went back to coding and combined a set of subsystems so that it could hear, think, and speak. Vision was also in development, but it wasn't connected to thinking. However, it could detect objects and display them in real time.
