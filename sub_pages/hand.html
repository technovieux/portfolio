Need a hand ?

More details

As part of my robot project, I wondered if it was possible to control it remotely so that my movements would be replicated. That's why I focused on the hands and how to control them using a computer.

The first part I had to focus on was controlling the artificial hand. This was made possible by servomotors and 3D printing. The hand is complete, and each finger can be controlled with a value between zero and 180°.

For the detection part, after some research, using Python and MediaPipe was the best solution. Then, mathematical functions calculate the space between each phalanx and calculate the angle of each finger to determine how straight or bent it is. All that remained was to assemble the two to control the artificial hand in real time.
