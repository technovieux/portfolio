Big Brother

Smile, you might be recorded

More details

During my time in Romania, I was asked to develop a Python program that used computer vision to count people. So I developed this project with a graphical user interface using CustomTKinter and two detection models: one using deep learning to detect full bodies and another using Haar cascades to detect faces. The results are then displayed with green and blue rectangles directly on the image.

During this project, I had to manage several aspects, notably the use of multithreading to prevent the interface from freezing during image processing. When processing a video, certain frames are intentionally skipped to reduce the computational load, which is very energy-intensive. The time it takes to display the image will also increase depending on the number of detected objects.

In addition to detection, it is also possible to track detections using counters that display the number of faces in real-time or since the start, as well as using Matplotlib to generate graphs for a more “surveillance-style” look. All this data can be exported as JSON or CSV for use in an external program.
